# GroupDocs.Markdown - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Markdown](https://docs.groupdocs.com/markdown/) static documentation.

## About GroupDocs.Markdown

[GroupDocs.Markdown](https://products.groupdocs.com/markdown/) is a document processing SDK that enables you to export documents to Markdown format, making your content gen AI ready.

Available for .NET.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/markdown/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
