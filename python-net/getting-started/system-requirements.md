{{< alert style="info" >}}

GroupDocs.Markdown for Python via .NET does not require any external software such as Microsoft Office or third-party document editors. To install GroupDocs.Markdown for Python via .NET, follow the steps in the [Installation]({{< ref "markdown/python-net/getting-started/installation.md" >}}) guide.

{{< /alert >}}

## Supported Operating Systems

### Windows

*   Microsoft Windows 10 (x64, x86)
*   Microsoft Windows 11 (x64)
*   Microsoft Windows Server 2016 and later

### Linux

*   Ubuntu 20.04+, Debian 11+, CentOS 8+, Fedora 36+, Alpine 3.16+

### macOS

*   macOS 12 (Monterey) and later -- Intel and Apple Silicon (M-series)

## Python Version

GroupDocs.Markdown for Python via .NET supports **Python 3.5 through 3.14**. The wheel uses the `py3-none-{platform}` tag, meaning it works with any Python 3.x version without per-version rebuilds.

| Python Version | Supported |
| --- | :---: |
| 3.5 | Yes |
| 3.6 | Yes |
| 3.7 | Yes |
| 3.8 | Yes |
| 3.9 | Yes |
| 3.10 | Yes |
| 3.11 | Yes |
| 3.12 | Yes |
| 3.13 | Yes |
| 3.14 | Yes |

## Package Manager

GroupDocs.Markdown for Python via .NET is distributed via [PyPI](https://pypi.org/project/groupdocs-markdown-net/):

```bash
pip install groupdocs-markdown-net
```
